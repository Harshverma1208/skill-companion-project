Frontend Guidelines

State Management: Use Redux or Context API.

Component Structure:

Pages: Dashboard, Skill Gap Analysis, Course Recommendations, Trend Analysis, Resume Analyzer, Profile

Components: Navbar, Footer, CourseCard, JobTrendCard, ResumeUpload, etc.

API Handling: Use Axios for API calls with proper error handling.

UI Framework: Tailwind CSS / Material UI for a modern design.

Accessibility: Ensure WCAG-compliant UI for better usability.
